# svelte-dnd Library

This package contains the publishable drag-and-drop runtime and its package tests.

## Scripts

```sh
bun install
bun run test
bun run check
bun run build
```

## Layout

- `src/lib/`
  Library source and tests.
- `src/routes/`
  Temporary local preview surface for package work.

The broader playground and future docs website live in `../site`.
